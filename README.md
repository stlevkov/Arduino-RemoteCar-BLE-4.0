# Arduino-RemoteCar-BLE-4.0
Arduino controlled remote car using BLUETOOTH Transiever HM-10

<h1> Hardware used in this project:</h1>

<ol>
  <li>Arduino Leonardo</li>
  <li>BLUETOOTH Transiever HM-10</li>
  <li>X,Y, Joystick</li>
  <li>9V Battery</li>
</ol>
